# MyTask Application

MyTask is a modern Android application designed to help you manage your daily tasks efficiently. Built with **Kotlin** and **Jetpack Compose**, it features a clean user interface and seamless navigation.

## Features

*   **Task Management**: Create, view, and manage your to-do list.
*   **User Authentication**: Secure Sign In and Sign Up screens.
*   **Map Integration**: Visualize tasks on a map (integration ready).
*   **Modern UI**: Built using Material Design 3 and Jetpack Compose.

## Tech Stack

*   **Language**: Kotlin
*   **UI Toolkit**: Jetpack Compose
*   **Navigation**: Jetpack Compose Navigation
*   **Build System**: Gradle (Kotlin DSL)

## Getting Started

1.  Clone the repository:
    ```bash
    git clone https://github.com/Kondareddy1209/Task-Application.git
    ```
2.  Open the project in **Android Studio**.
3.  Sync Gradle files.
4.  Run the application on an emulator or physical device.
